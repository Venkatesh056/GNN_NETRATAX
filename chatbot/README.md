# GST Fraud Detection Chatbot

A Streamlit-powered AI chatbot to answer questions over GST invoice and company data using OpenAI API.

## Setup
1. Place your GST datasets as `data/companies.csv` and `data/invoices.csv`
2. Install requirements: `pip install -r requirements.txt`
3. Add your OpenAI key in `.streamlit/secrets.toml`
4. Launch the chatbot: `streamlit run app.py`
